Echo Robotics Website

Project Overview

The purpose of this website is to showcase the Echo Robotics line of products. Echo Robotics develops really cool and oldy big robotic lawn mowers. These lawn mowers are built for commercial facilities, golf courses, and sports fields (I've asked and you are not allowed to ride them). This site has a multiple languages using the WPML plugin, a dealer resource center, and a blog. This site was built using the help of Elementor page builder and some custom Elementor elements that I created.
