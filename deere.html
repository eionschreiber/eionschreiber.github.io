John Deere Virtual Pavilion

Project Overview

The purpose of this website is to provide a virtual showroom like experience for John Deere. The site is separated into four groups, Agriculture, Landscaping, Sports Turf, and Golf. Each one of these categories explores a different market. This site was really fun to build, I along with another developer helped build this site. The site features 360 views of many, many products. It also features information of each of the products in the form of hotspots, and videos. It's a real show stopper for a website and I was proud to be apart of it.
